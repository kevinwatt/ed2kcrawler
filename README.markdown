
Install/Run Example
-------------------

    $ git clone git://github.com/kevinwatt/ed2kcrawler.git

Build ed2kcrawler

    $ make
    
Create an config.cfg file

    [GenSetting]
    OThread = 3

    [amule]
    ARS = 10.8.0.1
    ARP = 4712
    ARPS = amulepasswd

Running ed2kcrawler
    
    $ ./ed2kcrawler -l vv
